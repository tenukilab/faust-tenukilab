秘書（内部処理）：
- public/games/breakout/index.html を設計・作成
  - Canvas APIでブロック崩しを実装
  - てぬきラボブランドカラー（ピンク・オレンジ・イエロー・グリーン・パープル）でブロック配色
  - パドル：マウス・矢印キー・タッチ操作対応
  - レベルアップ機能（全消しで次レベル、スピード上昇）
  - ライフ制（3回）
  - ゲームオーバー文言：「クロちゃんが『もっと頑張れ』と言っています。」
  - クリア文言：「てぬきさんが手を動かしたかもしれない。」
  - 実験室へ戻るリンク付き

- Lab.jsxのlabItemsに登録
  - タイトル：てぬきブロック崩し
  - ステータス：公開中
  - タグ：ゲーム、Canvas、JavaScript
  - リンク：/games/breakout/

- npm run build → ビルド成功
- netlify deploy --prod → デプロイ成功
